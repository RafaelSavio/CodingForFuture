What is CSS ?

CSS is an abbreviation of Cascading Style Sheet, which is a web document where its funtion is to manage the HTML elements with different properties that are provided, so the project will have a variety of aesthetics. CSS is not considered as a programming language as it have a simple structure which is a compilation of configurations to adjust the HTML elements.

Selector

Selector is the term used to select a certain HTML element which will be modified using CSS. In general, there are three types of selectors in CSS, which are class selector, id selector and tag selector.

Tag Selector

Tag selector is a selector which directly uses the HTML tag name that want to be altered or modified by CSS. However in this case the HTML tag does not use the < > sign

Selector Class

Selector class is a selector which uses a class name used in a tag that wants to be configured by CSS. However the class name is accompanied by a (.), placed before the name. This selector’s function is more specific than the tag selector.

Id Selector

Selector id is a selector form which uses the id name from an HTML element which wants to be modified by CSS. In this case, the id name is accompanied by a hashtag (#) before the id name. However this selector can only be used for one HTML element, which makes this selector more specific than the other two.

Property and Value

Property is the characteristics or rules that is given to an HTML element such as, color, background, font and so-on. In writing specific properties, there will be Value, which creates the value of a property. Property is a selector which is placed between the ( and ) sign. Property and value are separated by a colon (:) and ended with a “;” sign. Every value of a property is different, there are some with values in form of numbers and some in values that are available for a certain property.
Almost all property in CSS has a derivative, for example, the property background has a derivative of background-image, background-repeat, background-position and etc.

Absolute Units

Absolute units are units with a fixed size , unchanged and unaffected by any elements. Absolute units include, cm (centimeter), mm (millimeter), in (inch), px (pixel), pt (point), and pc (pica). The ratio between each units are:
1in = 96 px = 2,54cm
1px = 1/96in
1pt = 1/72in
1pc = 12pt

Box Model

Box model is a concept where every element on every webpage is considered as a box. This box will then build the layout and design structure of a website. This concept is very much attached to every element in HTML.

Height and Width

The height property determines the height of the box, whereas the width property determines the width of the box. If a background is given a <div> tag with CSS, normally it wouldn’t show up if the width and height is not determined yet. The value from this property is in the form of numbers with the units being px or %.
There are other properties which are related to width and height, such as:
Max-width, to configure the width which adjusts the content but not over the determined border
Max-height, to configure the height to adjust the content but not over the determined border
Min-height, to configure the height to adjust the content but not less than the determined border
Min-width, to configure the width to adjust the content but not less than the determined border

Margin and Padding

The margin property configures the distance of the box from the elements around it or the wrapping elements of the box (container). Whereas, the padding adjusts the distance between the box and the contents which is within the box. The margin property have several derivatives, such as margin-top, margin-right, margin-bottom, and margin-left. As for padding its derivatives consist of, padding-top, padding-right, padding-bottom and padding-left. The four derivatives of both padding and margin can be simplified to the margin [top] [right] [bottom] [left] and padding [top] [right] [bottom] [left]. The values of both properties are in the form of numbers with a general unit which is px. The other values that can be given are inherit and auto. If it is auto, the browser will create all values having the same values on each side, therefore creating an equal box.

Border and Outline

The border property creates a frame for the box. The border property have derivatives that are divided into two. The first part shows the border sides which consist of, border-top, border-right, border-bottom and border-left. The second part configures the style of the border, it consist of, border-width, border-style and border-color. Both parts can be combined into: border-top-style, border-left-style, border-top-color, border-left-color, border-top-size and etc. If the first derivative type is combined, the arrangement is the same as margin and padding.
There is a similar property to border, which is outline, the derivative and configuration of the property usage is the same as border, including the inputted property values. The difference is that outline is not an element, which means it does not affect the size of the box. In some cases, outline may cover the other contents. The outline property has one derivative that the border property does not, which is, outline-offset. Outline offset is the derivative which configures the distance between the outline and the box elements.
