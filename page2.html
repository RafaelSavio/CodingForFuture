HTML

What is HTML ?

HTML is an abbreviation of Hyper Text Markup Language, which is a standardized web language where its funtionality is monitored by the W3C (World Wide Web Consortium). The language is made up of tags which assembles every element of a website. HTML has a role of creating the structure of a webpage which places every element based on the plotted layout.

HTML is usually saved in a ".html" format. To create or start a HTML script, users usualy use a text editor such as notepad++, sublime text or any other text editor softwares that could display different colors to create an easier usage.



Elements of HTML

1. Tag

Tag is special symbol which includes two characters which are < and > that staples around a text as a tag name. For example <body> is a tag with a tag name, body. Most tags have opening tags and closing tags to limit the effect of the tag. However, there are tags that do not have closing tags. Closing tags is similar to a normal tag however, a "/" is included after the "<". For example, </body>.

1.1. Characteristics of tags in HTML

Tags can be writen in capital letters because its incase sensitive. However, it is reccomended to use lowercase letters.

Text and Paragraph formatting

Paragraph Formatting

Paragraphs are made with the tags, <p>...</p>. The attribute that is most commonly used is align. This attribute functions to organize the layout of the paragraph. The values that can be inputted in this attribute are, left, right, center and justify. Every <p> tag used, it will form a new paragraph creating a spacing. If you want to create a spacing without using the <p> tag, you can use <br> without a closing tag.

Text Style

The text created in HTML can be made bold, italic, underlined, striked-through and so on. These are the tags used to format the text style:
Tag <b> … </b>, to create a bolded text
Tag <i> … </i>, to italicize a text
Tag <u> … </u>, to underline a text
Tag <strike> … </strike> to make a striked-through text
Tag <strong> … </strong>, to show an important text by making it bold
Tag <em> … </em>, to show an important text by making it italicized
Tag <small> … </small>, to create a smaller sized text than normal
Tag <mark> … </mark>, to highlight a text
Tag <del> … </del>, to show an erased text with a similar effect of <strike> … </strike> tag.
Tag <ins> … </ins> to show an inserted text with an underline.
Even though tag <b> and tag <strong>, <strike> and <delete>, tag <i> and tag <em>,and tag <u> and tag <ins> have the same effect. The tags <strong> and <em> is more to pointing out a more important text.

Adding Image

The tag used to add an image on your html script is <img>. This tag does not have a closing tag and the attributes available for this tag are:
Src, to determine the source and the name of the image
Width, to determine the width of the image
Height, to determine the height of the image
Alt, to determine the height of the image
Title, to create a text alternative which displays, whenever the mouse is hovered on top of the image
If the image height and width is not determined, therefore, the image size is the original format. However, if the height and width is determined the image will not be proportional. If however the size wants to be changed, it is best to determine the height or the width only.
The image format that is supported by the tag <img> are jpg, png and gif. The name and image format must be written in full on the src attribute. There are different cases in adding an image which are:
If the image is one folder with the HTML file, therefore, the form of writing the attribute is src=”image_name.jpg”.
If the image is located in a different folder from the HTML file, then the writing format will be src=”folder_name/image_name.jpg”.
If the image file is in another file folder with another HTML file, it must be transferred outside of the folder, therefore to do so it must be written like so: src=”../folder_name/image_name.jpg”. The sign “../” means it exits the folder that has the other HTML file.

Semantic HTML 5 template

Before HTML5, there used to be an HTML template which us made with <div> tag which is given an id or class header, nav, menu, content, title, and footer. On the new HTML 5, there are new tags that are introduced which represents website sections , with the purpose of conveying the purpose of the elements to the developer, browser and algorithm search engine.
The added tags are:
The <header>...</header> tag, used to show the parts that function as a website header. This tag can be used several times based on the needs. Even if it is inside a content, the <header> tag can be included.
The <nav>...</nav> tag, used to show the parts used for the website navigation.
The <section>...</section> tag, used to separate parts of the website content. This tag can be used repeatedly in one webpage.
The <main>...</main> tag, used to display the most important parts of a webpage. Different with the <section> tag, this tag is normally used once on the most important part, such as, the content/main article.
The <article>...</article> tag, used to contain contents that are in the form of an article. This tag is usually used within the <main> or <section> tag.
The <aside>...</aside> tag, used to sow the parts that are not the main but contents that have a relation with the main content being displayed. This tag is usually used to create a sidebar.
The <footer>...</footer> tag, used to create a footer for the webpage, which is placed on the bottom.
